# 🎵 PULSE | Premium Audio Experience

**Pulse** is a state-of-the-art, high-fidelity music application designed for a premium listening experience. Built with a focus on **Black Luxury** aesthetics and cinematic animations, Pulse transforms your local music library into a gallery of sound.

## ✨ Key Features

- **🏆 Award-Winning UI/UX**: A "Black Luxury" design system featuring floating glass panels, gold/champagne accents, and frosted glass effects.
- **✨ Dynamic Cover Art**: Algorithmic generation of beautiful, unique cover art for every track based on its metadata.
- **🌊 Cinematic Visualizer**: A high-performance, golden-glowing audio spectrum that reacts in real-time to your music.
- **🌌 Ambient Aurora**: A slow-moving, atmospheric background light that provides depth and motion.
- **🛠️ Smart Queue Management**: Slide-out "Up Next" panel with one-click track navigation.
- **💨 Blazing Fast**: Minimalist vanilla JavaScript core with a high-performance FastAPI backend.

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Modern Web Browser (Chrome/Edge/Safari recommended for backdrop-filter support)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/pulse.git
   cd pulse
   ```

2. **Set up the Backend**
   ```bash
   cd backend
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

3. **Launch the Frontend**
   Simply open `frontend/index.html` in your browser or use a Live Server.

## 🎨 Design Philosophy

Pulse was designed using a **"Floating Island"** architecture. Unlike traditional music players, every UI component in Pulse is a detached glass panel, creating a sense of weightlessness and sophistication. The use of a **Film Grain Overlay** and **Bezier-curved animations** ensures that the interface feels tactile and alive.

## 🛠️ Tech Stack

- **Frontend**: Vanilla JS, HTML5, CSS3 (Glassmorphism, CSS Keyframes)
- **Backend**: FastAPI (Python), SQLite
- **Visuals**: Canvas API (Real-time Audio Processing)
